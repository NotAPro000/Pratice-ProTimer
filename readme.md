## Development Environment

###  Controller (MCU) 
- PIC16F18877

### IDE
- MPLAB X IDE V6.2.0
- XC8 V3.0.0 Compiler
- MCC V5.5.0

## MCU Config/Pin Defined
|Nname       |Parameter|
|------------|------------|
|**Clock Source**|External oscillator|
|**Clock Frequency**|12MHz|
|**External Oscillator Mode**|HS|
|**LCD Reset Bit**|RC0|
|**LCD Enable Bit**|RC1|
|**LCD Data Bits**|RD0~RD7|
